# IT Tech Knowledge Base

Cette base de connaissances est un parcours pratique pour devenir un bon technicien ou une bonne technicienne en soutien informatique de niveau debutant a intermediaire.

Elle ne sert pas seulement a memoriser des definitions. Chaque fichier vise a t'apprendre a :

1. comprendre le systeme avant de le modifier;
2. diagnostiquer un probleme de facon structuree;
3. utiliser les bons outils et commandes;
4. proteger les donnees et eviter les changements risques;
5. communiquer clairement avec l'utilisateur;
6. documenter et escalader correctement;
7. expliquer tes decisions en entrevue.

## Comment utiliser cette base

- Lis les dossiers dans l'ordre, en commencant par `01-networking`.
- Ne lis pas plus de deux ou trois fichiers sans pratiquer.
- Execute les commandes dans un laboratoire, une machine virtuelle ou ton propre environnement non critique.
- Apres chaque fichier, complete les exercices et reponds aux questions sans regarder les reponses.
- Ajoute tes propres captures, erreurs et solutions dans `11-my-projects/lessons-learned.md`.
- Utilise `PRACTICE-TRACKER.md` pour suivre ta progression.

## Niveau attendu

A la fin, tu devrais pouvoir :

- depanner un poste Windows, une connexion reseau, Outlook, une imprimante et un VPN;
- effectuer les taches d'entree de gamme dans Active Directory et Microsoft 365;
- lire des journaux Windows et Linux;
- utiliser PowerShell et les commandes Linux de base;
- appliquer les principes de securite, de sauvegarde et de moindre privilege;
- prendre en charge un ticket, communiquer avec l'utilisateur, documenter puis escalader;
- presenter ton homelab Ollama/OpenWebUI et Tailscale comme experience pratique.

## Regle de securite

Ne teste jamais une commande destructive, une modification de registre, une strategie de groupe ou une permission sur un environnement de production sans autorisation, sauvegarde et plan de retour arriere.
