# IT Tech Knowledge Base

Cette base de connaissances est un parcours pratique pour devenir un bon technicien ou une bonne technicienne en soutien informatique de niveau débutant à intermédiaire.

Elle ne sert pas seulement à mémoriser des définitions. Chaque fichier vise à t'apprendre à :

1. comprendre le système avant de le modifier;
2. diagnostiquer un problème de façon structurée;
3. utiliser les bons outils et commandes;
4. protéger les données et éviter les changements risqués;
5. communiquer clairement avec l'utilisateur;
6. documenter et escalader correctement;
7. expliquer tes décisions en entrevue.

## Comment utiliser cette base

- Lis les dossiers dans l'ordre, en commençant par `01-networking`.
- Ne lis pas plus de deux ou trois fichiers sans pratiquer.
- Exécute les commandes dans un laboratoire, une machine virtuelle ou ton propre environnement non critique.
- Après chaque fichier, complète les exercices et réponds aux questions sans regarder les réponses.
- Ajoute tes propres captures, erreurs et solutions dans `11-my-projects/lessons-learned.md`.
- Utilise `PRACTICE-TRACKER.md` pour suivre ta progression.

## Niveau attendu

À la fin, tu devrais pouvoir :

- dépanner un poste Windows, une connexion réseau, Outlook, une imprimante et un VPN;
- effectuer les tâches d'entrée de gamme dans Active Directory et Microsoft 365;
- lire des journaux Windows et Linux;
- utiliser PowerShell et les commandes Linux de base;
- appliquer les principes de sécurité, de sauvegarde et de moindre privilège;
- prendre en charge un ticket, communiquer avec l'utilisateur, documenter puis escalader;
- présenter ton homelab Ollama/OpenWebUI et Tailscale comme expérience pratique.

## Règle de sécurité

Ne teste jamais une commande destructive, une modification de registre, une stratégie de groupe ou une permission sur un environnement de production sans autorisation, sauvegarde et plan de retour arrière.
